# 📋 Tracker Semanal — Calendário Estratégico

Aplicativo de rastreamento semanal de atividades construído em React + Vite, projetado para empreendedores que gerenciam múltiplas frentes de trabalho simultaneamente.

## Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/import/project)

**Framework Preset:** Vite  
**Build Command:** `npm run build`  
**Output Directory:** `dist`

## Setup local

```bash
npm install
npm run dev
```

## Stack

React 18 · Vite 6 · localStorage · HTML5 Drag & Drop
